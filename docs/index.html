Running RingdateR in Rstudio

To use RingdateR in Rstudio, download the ringdater_0.1.0.tar.gz file. In Rstudio, on the package tab, click install packages. Change the "Install From" option from "Repository (CRAN)" to "Package Archive File (.tgz, .tar.gz)". Use the "Package Archive" browser to find and select the ringdater_0.1.0.tar.gz file you just downloaded. Click install.

RRunning RingdateR online

Use the link above to launch the online version of RingdateR. Please note though, that we strongly recommend you download and run RingdateR locally if you decide to use it regularly. There is limited access to the online version.
